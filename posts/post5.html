Thread Pool trong Java

Thread Pool là kỹ thuật tạo sẵn một nhóm thread để xử lý các công việc. Điều này giúp tiết kiệm tài nguyên hệ thống.

Khi có yêu cầu mới, thread rảnh trong pool sẽ được sử dụng. Sau khi xử lý xong, thread quay lại pool.

Thread Pool thường được dùng trong server xử lý nhiều kết nối cùng lúc.

Tài liệu tham khảo: Java Concurrency
