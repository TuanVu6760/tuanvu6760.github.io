Fetch API trong JavaScript

Fetch API cho phép JavaScript gửi và nhận dữ liệu từ server thông qua giao thức HTTP.

Fetch có cú pháp đơn giản và dễ sử dụng hơn XMLHttpRequest.

Fetch API thường được dùng để gọi API backend và xử lý dữ liệu động.
